# 快速安装卡片

一页纸了解插件安装

---

## 📦 安装包

**文件名**: `lingyun-openclaw-plan-manager-1.0.0.tgz`
**大小**: 44KB
**包含**: 9个计划管理工具

---

## 🚀 3步安装 (Windows/Linux)

```bash
# 1. 安装
openclaw plugins install lingyun-openclaw-plan-manager-1.0.0.tgz

# 2. 配置
cd ~/.openclaw/plugins/@lingyun/openclaw-plan-manager
cp .env.example .env
# 编辑 .env 填入后端地址和Token

# 3. 重启
openclaw gateway restart
```

✅ 完成!在 OpenClaw 中说 "帮我制定一个计划"

---

## 🍎 macOS 特别提示

如果遇到 `openclaw: command not found`:

```bash
# 使用完整路径
/Applications/OpenClaw.app/Contents/MacOS/openclaw plugins install lingyun-openclaw-plan-manager-1.0.0.tgz

# 或手动安装 - 见 macOS安装指南.md
```

---

## ⚙️ 必需配置

在 `.env` 文件中填入:

```env
AI_PLAN_API_BASE=http://服务器地址:4000/api
AI_PLAN_API_TOKEN=你的Token
```

**获取 Token**:
1. 登录 AI计划管理系统
2. 个人设置 → OpenClaw集成
3. 生成Token → 复制
