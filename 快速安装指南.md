# 快速安装指南

## 🚀 一键安装 (推荐)

### Windows 用户:
```cmd
cd openclaw-plan-manager
install.bat
```

### Linux/Mac 用户:
```bash
cd openclaw-plan-manager
chmod +x install.sh
./install.sh
```

脚本会自动完成所有步骤!

---

## 📝 手动安装步骤

如果自动安装失败,按以下步骤操作:

### 步骤1: 配置环境变量

```bash
# 复制环境变量文件
copy .env.example .env  # Windows
# cp .env.example .env  # Linux/Mac
```

编辑 `.env` 文件:
```env
AI_PLAN_API_BASE=http://localhost:4000/api
AI_PLAN_API_TOKEN=your_token_here
AI_PLAN_TIMEOUT_MS=15000
```

### 步骤2: 安装依赖

```bash
npm install
```

### 步骤3: 编译 TypeScript (重要!)

```bash
npm run build
```

这会将 TypeScript 代码编译到 `dist/` 目录。

### 步骤4: 安装到 OpenClaw

```bash
openclaw plugins install -l .
```

### 步骤5: 重启 Gateway

```bash
openclaw gateway restart
```

### 步骤6: 验证安装

```bash
openclaw plugins list
openclaw tools list
```

---

## ⚠️ 重要说明

1. **必须先编译**: OpenClaw 无法直接运行 TypeScript 文件,必须先编译成 JavaScript
2. **dist 目录**: 编译后的文件在 `dist/` 目录,这是 OpenClaw 实际加载的文件
3. **main 字段**: package.json 中 `main` 指向 `dist/index.js`

---

## 🔧 开发模式

如果要修改代码:

```bash
# 1. 修改 TypeScript 源码
# 2. 重新编译
npm run build

# 3. 不需要重新安装,因为使用了 -l 链接模式
# 4. 重启 Gateway 加载新代码
openclaw gateway restart
```

## 📦 打包发布

准备发布到 ClawHub:

```bash
# 1. 编译
npm run build

# 2. 打包 (会自动运行 prepublishOnly 脚本)
npm pack

# 3. 生成文件
# lingyun-openclaw-plan-manager-1.0.0.tgz

# 4. 发布
openclaw publish lingyun-openclaw-plan-manager-1.0.0.tgz
```

## 🐛 故障排查

### 问题: "Cannot find module 'dist/index.js'"

**原因**: 忘记编译

**解决**:
```bash
npm run build
```

### 问题: 编译错误

**解决**:
```bash
# 检查依赖
npm install

# 清理重新编译
rm -rf dist
npm run build
```

### 问题: 插件加载失败

**检查**:
1. dist/ 目录是否存在?
2. dist/index.js 文件是否存在?
3. package.json 中 main 是否指向 dist/index.js?
