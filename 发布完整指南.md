## 发布前准备

### ✅ 检查清单

在开始发布之前，请确认以下事项：

- [ ] 所有代码已测试通过
- [ ] 插件在本地 OpenClaw 中运行正常
- [ ] 所有文档文件已创建（README.md、INSTALLATION.md、CHANGELOG.md 等）
- [ ] package.json 版本号正确（1.0.0）
- [ ] 已准备好 GitHub 账号
- [ ] 已准备好个人邮箱

### 📝 需要替换的占位符

在发布之前，需要将以下占位符替换为实际值：

| 占位符 | 需要替换成 | 在哪些文件中 |
|--------|-----------|-------------|
| `YOUR_USERNAME` | 你的 GitHub 用户名 | 所有 .md 文件、package.json、claw-hub.json |
| `your-email@example.com` | 你的真实邮箱 | package.json、claw-hub.json、README.md |

#### 批量替换方法

**Windows PowerShell：**
```powershell
# 进入插件目录
cd "F:\菩提闻道\001-塞外飞雪2024\桌面\AI计划管理系统_网页版20260328\openclaw-plan-manager"

# 替换 GitHub 用户名（将 your-github-username 改为你的用户名）
Get-ChildItem -Recurse -Include *.md,*.json | ForEach-Object {
    (Get-Content $_.FullName) -replace 'YOUR_USERNAME', 'your-github-username' | Set-Content $_.FullName
}

# 替换邮箱（将 your.email@gmail.com 改为你的邮箱）
Get-ChildItem -Recurse -Include *.md,*.json | ForEach-Object {
    (Get-Content $_.FullName) -replace 'your-email@example.com', 'your.email@gmail.com' | Set-Content $_.FullName
}
```

**macOS/Linux：**
```bash
# 进入插件目录
cd "/path/to/openclaw-plan-manager"

# 替换 GitHub 用户名
find . -type f \( -name "*.md" -o -name "*.json" \) -exec sed -i '' 's/YOUR_USERNAME/your-github-username/g' {} +

# 替换邮箱
find . -type f \( -name "*.md" -o -name "*.json" \) -exec sed -i '' 's/your-email@example.com/your.email@gmail.com/g' {} +
```

### 🔨 最后构建

确保插件已正确构建：

```bash
cd openclaw-plan-manager
npm install
npm run build
```

确认 `dist/` 目录已生成且包含编译后的 JavaScript 文件。

## 发布到 GitHub

### 步骤 1: 创建 GitHub 仓库

1. **登录 GitHub**
   - 访问 [https://github.com](https://github.com)
   - 登录你的账号

2. **创建新仓库**
   - 点击右上角的 `+` → `New repository`

3. **填写仓库信息**
   - **Repository name**: `lingcloud-ai-plan-manager`
   - **Description**: `A powerful OpenClaw plugin for intelligent plan and task management with AI assistance`
   - **Public/Private**: 选择 `Public`（公开仓库）
   - **不要勾选** "Initialize this repository with a README"（我们已经有了）
   - **不要选择** .gitignore 和 license（我们已经有了）

4. **点击 "Create repository"**

### 步骤 2: 初始化本地 Git 仓库

打开命令行，执行以下命令：

```bash
# 进入插件目录
cd "F:\菩提闻道\001-塞外飞雪2024\桌面\AI计划管理系统_网页版20260328\openclaw-plan-manager"

# 初始化 Git 仓库
git init

# 添加所有文件到暂存区
git add .

# 创建第一次提交
git commit -m "🎉 Initial release v1.0.0

Features:
- Plan management with AI assistance
- Task tracking and status updates
- Execution reports and analytics
- Smart reminders and notifications
- Time range filtering (week, last_week, month, all)
- 9 powerful tools for complete workflow
- Natural language interaction
- Support for hierarchical tasks

This is the first stable release of LingCloud AI Plan Manager plugin."
```

### 步骤 3: 连接到 GitHub 仓库

将本地仓库连接到 GitHub（**替换 `your-github-username` 为你的实际用户名**）：

```bash
# 添加远程仓库
git remote add origin https://github.com/your-github-username/lingcloud-ai-plan-manager.git

# 设置默认分支为 main
git branch -M main

# 推送到 GitHub
git push -u origin main
```

如果推送时需要认证：
- **用户名**: 你的 GitHub 用户名
- **密码**: 使用 GitHub Personal Access Token（不是账号密码）

**如何获取 Personal Access Token：**
1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Generate new token
3. 勾选 `repo` 权限
4. 生成并复制 token
5. 将 token 作为密码使用

## 发布到 Claw Hub

### 前提条件

- ✅ GitHub 仓库已创建并公开
- ✅ v1.0.0 Release 已发布
- ✅ `claw-hub.json` 文件已准备好

### 步骤 1: 验证 claw-hub.json

确保所有信息正确（特别是 GitHub 链接和邮箱）：

```json
{
  "name": "lingcloud-ai-plan-manager",
  "version": "1.0.0",
  "repository": {
    "url": "https://github.com/your-github-username/lingcloud-ai-plan-manager.git"
  },
  "author": {
    "email": "your-email@example.com"
  }
}
```

### 步骤 2: 提交到 Claw Hub

#### 方法 A: 通过 OpenClaw CLI

```bash
cd openclaw-plan-manager
openclaw publish claw-hub.json
```

#### 方法 B: 通过 Claw Hub 网站

1. 访问 Claw Hub 网站
2. 登录账户
3. 点击 "Submit Plugin"
4. 填写表单并上传 `claw-hub.json`
5. 提交审核

### 步骤 3: 等待审核

- 审核通常需要 1-3 个工作日
- 审核通过后会收到邮件通知

## ❓ 常见问题

### Q: GitHub 推送要求密码？

**A:** 使用 Personal Access Token：
- GitHub → Settings → Developer settings → Personal access tokens
- Generate new token → 勾选 `repo`
- 用 token 作为密码

### Q: 如何验证发布成功？

**A:** 在新目录测试克隆：
```bash
git clone https://github.com/your-username/lingcloud-ai-plan-manager.git
cd lingcloud-ai-plan-manager
npm install && npm run build
```

### Q: 用户安装失败？

**A:** 常见原因：
- Node.js < 18 → 升级到 18+
- 网络问题 → 使用代理
- 权限问题 → 使用管理员权限
