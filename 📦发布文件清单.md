## 🚀 发布步骤概览

### 1️⃣ 准备阶段（5分钟）
```powershell
# 替换占位符
Get-ChildItem -Recurse -Include *.md,*.json | ForEach-Object {
    (Get-Content $_.FullName) -replace 'YOUR_USERNAME', 'your-github-username' | Set-Content $_.FullName
}

# 构建插件
npm install
npm run build
```

### 2️⃣ 发布到 GitHub（10分钟）
1. 创建 GitHub 仓库：`lingcloud-ai-plan-manager`
2. 推送代码
3. 创建 Release：`v1.0.0`

### 3️⃣ 发布到 Claw Hub（5分钟）
```bash
openclaw publish claw-hub.json
```

### 4️⃣ 验证发布
```bash
git clone https://github.com/你的用户名/lingcloud-ai-plan-manager.git test
cd test && npm install && npm run build
```
