## 🌐 发布到 GitHub（10分钟）

### Step 1: 创建仓库
1. 访问 https://github.com
2. 点击 `+` → `New repository`
3. 名称：`lingcloud-ai-plan-manager`
4. 选择 `Public`
5. 不要初始化 README

### Step 2: 推送代码
```bash
cd openclaw-plan-manager
git init
git add .
git commit -m "🎉 Initial release v1.0.0"
git remote add origin https://github.com/你的用户名/lingcloud-ai-plan-manager.git
git branch -M main
git push -u origin main
```

### Step 3: 创建 Release
1. GitHub 仓库页面 → `Releases` → `Create a new release`
2. Tag: `v1.0.0`
3. Title: `🎯 LingCloud AI Plan Manager v1.0.0`
4. 复制 `发布完整指南.md` 中的 Description
5. 点击 `Publish release`

✅ GitHub 发布完成！
